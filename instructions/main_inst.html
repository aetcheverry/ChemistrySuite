Chemistry Suite

General Help & Documentation

Chemistry Suite is a modular software platform designed to assist with common laboratory calculations and data referencing. The software is divided into two main sections: Data (reference materials) and Calculators (interactive tools).

1. Navigation
Getting Around

Use the Side Panel on the left to navigate between different modules.

  • Clicking a category header (e.g., "Calculators") expands or collapses that section.
  • Clicking a specific module (e.g., "Elemental Analysis") loads that tool in the main view.
Tip: Context-specific instructions for each tool can be accessed by pressing F1 while the tool is open.
2. Menu Bar Features
File Menu
  • Preferences... (Ctrl+P): Open the configuration panel to customize the application appearance (theme, font size), define startup behaviors, manage export settings, and set default file paths.
  • Home (Ctrl+H): Returns to the main landing page.
  • Exit (Alt+F4): Closes the application.
Edit Menu
  • Clear Inputs (Ctrl+X): Resets all text fields and selections in the currently active module.
  • Copy Results (Ctrl+C): Copies raw calculation results (e.g., tables) to your clipboard for use in Excel or other software.
  • Copy Report (Ctrl+Shift+C): Copies the formatted text report from the active module to your clipboard.
  • Export Plot... (Ctrl+E): Saves the current graphical plot (if available) as an image file.
Help Menu
  • Instructions (F1): Opens this help window or the specific help file for the active module.
  • Check for updates: Connects to the server to verify if a newer version of Chemistry Suite is available.
  • About Chemistry Suite: Displays version and developer information.
