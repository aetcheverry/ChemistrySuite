Solvent Properties

Physical Constants & Safety Data

The Solvent Properties module is a quick-reference database containing physical, chemical, and safety data for common laboratory solvents. It is designed to assist in reaction planning, workups, and solvent selection.

1. Search & Sort
Finding Solvents

The database contains a wide range of solvents. You can filter and organize this list to find the specific properties you need.

Controls:
  • Search Bar: Filter the list instantly by typing the solvent name or chemical formula (e.g., "DCM", "Acetone", "C6H6").
  • Sorting: Use the dropdown menus to reorder the list:
    • Sort By: Choose a property (e.g., Boiling Point, Density, Dielectric Constant).
    • Order: Toggle between Ascending (Low → High) and Descending (High → Low).
2. Customizing the View
Column Visibility

To keep the interface clean, not all data is shown by default. You can toggle specific columns on or off using the checkboxes in the settings area.

Available Properties:
  • Density: Mass per unit volume (in g/mL).
  • Viscosity: Resistance to flow (in mPa·s at 20 °C).
  • Vapour pressure: Pressure exerted by the solvent vapor in equilibrium with its liquid phase (in kPa).
  • Melting point: Solid-liquid phase transition (in °C).
  • Boiling point: Liquid-gas phase transition (in °C).
  • Flash point: Lowest temperature at which vapors ignite (in °C).
  • UV cutoff: Cutoff wavelength, below which the solvent absorption becomes excessive (in nm).
  • Refractive Index: Measure of light propagation speed (at 20 °C, stated otherwise).
  • Dielectric constant: Measure of solvent polarity (at 20 °C, stated otherwise).
  • Drying Agents: Recommended chemicals for drying the solvent.
Note: Empty cells mean no data available.
3. Utility Features
Export & Reset
  • Copy Table (Ctrl+C): Copies the currently visible rows and columns to the clipboard as tab-separated values (TSV), ready to paste into Excel or a lab notebook.
  • Clear Inputs (Ctrl+X): Resets the search filter, restores default sorting, and resets column visibility to the default preset.
