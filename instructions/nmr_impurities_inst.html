NMR Impurities Reference

Trace Impurities in Deuterated Solvents

This module provides a comprehensive database of chemical shifts for common laboratory impurities (solvents, grease, water) in various deuterated solvents. It is based on the standard data published by Fulmer et al.

1. Browsing the Data
Navigation & Filters

The interface allows you to toggle between Proton (1H) and Carbon (13C) data and filter the large table to find exactly what you need.

Controls:
  • Nucleus Toggle: Switch between Proton (1H) and Carbon (13C) modes using the radio buttons at the top.
  • Search Impurity: Type in the search box to filter rows by the impurity name (e.g., "Ethyl acetate", "Water", "Grease").
  • Filter by Solvent: Use the dropdown to isolate a specific deuterated solvent (e.g., CDCl3). This hides all other solvent columns for easier reading.
2. Reading the Table
Column Definitions

The main data table displays chemical shifts in ppm (δ).

  • Impurity: The name of the trace compound.
  • Mult (1H mode only): The multiplicity of the signal (e.g., s = singlet, t = triplet, q = quartet, m = multiplet).
  • Solvent Columns (e.g., CDCl3, DMSO-d6): The chemical shift value of the impurity in that specific solvent.
Empty Cells: A blank cell indicates that data is not available for that specific impurity/solvent combination, or the shift is obscured.
3. Notes & Citations
Footnotes & References

The panel at the bottom of the window displays essential context for the data.

  • Letter Codes: The letter codes are footnotes related to some impurities, such as the presence of isotopomers or the presence of additional resonances.
