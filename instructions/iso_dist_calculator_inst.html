Isotopic Distribution

Mass Spectrometry Simulation

This module simulates the theoretical isotopic distribution pattern for a given molecular formula. It is designed to help verify High-Resolution Mass Spectrometry (HRMS) data by predicting the exact m/z values and relative abundances of isotopic peaks.

1. Simulation Setup
Defining the Species

Configure the molecule and ionization parameters to match your experiment.

Steps:
  1. Enter the Chemical Formula (e.g., C20H25N3O).
  2. Ionization Mode: Select Positive (+) or Negative (-). This updates the list of available adducts.
  3. Adduct: Choose the adduct formed during ionization (e.g., [M+H]+, [M+Na]+, [M-H]-). The calculator automatically adjusts the formula and charge.
  4. Click Calculate Distribution.
2. Analysis Results
Table & Plot

The module generates two views of the data:

  • Peak Table: Lists the calculated m/z and relative Abundance (%) for the most significant isotopes (up to 99.9% coverage).
  • Spectral Plot: A visual representation of the mass spectrum (stick plot).
Utility Features
Export Options
  • Copy Results (Ctrl+C): Copies the Peak Table data (m/z and Abundance) to the clipboard as text.
  • Export Plot (Ctrl+E): Opens a dialog to save the current spectral plot as an image file (PNG, PDF, SVG, etc.) for use in publications or presentations.
